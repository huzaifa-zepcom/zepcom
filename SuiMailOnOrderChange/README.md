# SuiMailOnOrderChange

### A plugin for [Shopware 6](https://github.com/shopware/platform)

Plugin that gives you the chance to preview and change the mail that is going to be sent on order
status change.

## Requirements

| Version | Requirement     |
|-------- |---------------- |
| 1.0.0   | Shopware 6.4 >= |
